Spinnyt

2024

A Hypercasual mobile game that takes the the fast paced gameplay of Super Hexagon and adds in themed levels for spice.

The game was solo developed by yours truly! from the UI/UX to the game mechanics. I wanted to achieve an ergonomically pleasing look for the game that isn't hard on the eye and pushes the player to keep saying to themselves "one more try".

It's pretty simple (as are all Hypercasual games), the player must spin to avoid danger, whether that's piano notes attacking you in the "classical music" level or whatever the "punk" level has to offer. The longer you spin.. The better your score!

With a backend coded in NodeJS using MongoDB and deployed on render.com, the player's score is uploaded to a database, and the highest scores for each given level are provided.


Developed using

  • Unity3D
  • NodeJS
  • Visual Studio Code
  • C#
  • Android

Zwoop

2024

This was a passion project of mine. While developing Spinnyt, i noticed that my little brother loved hanging out to test every new update i made. And so i challenged myself to create a game for him, but do it in as little time as possible, while also delivering an addicting gameplay loop.

I've had the idea for a while, to make a game where you don't necessarily control a character, but you control your mouse, and your cursor must be the one dodging the obstacles.

I don't like to say that it's finished because now and again, i'd have a great idea that makes me go back to it and enhance it, whether it's stylistically or mechanically. But for now, it's more or less "finished".

For example, one of the additions i've added was the ability to use the mouse wheel to zoom in and out, thus shifting the perspective, to either see the incoming obstacles or to get small enough to avoid them.


Developed using

  • Unity2D
  • Visual Studio Code
  • C#

Interplanetary Ball

Work in progress

This single project so far has been the culmination of my game development journey and i have enjoyed every bit of it

Local star skybox

The stars that you're seeing aren't actually part of the skybox. Instead, they're a particle system emitting a large number of very small particles at the start of the game. This particle system is local to the player, but is rendered behind every other object in the scene. I wanted to recreate this trick after seeing it in my favorite game of all time Outer Wilds

Modifiable faux-gravity

I have put in place a system that mimmicks Super Mario Galaxy's faux gravity but on a planetary scale, where the sun attracts any and all objects, thus bending the ball's trajectory.

Realistic Sun Shader

Created a very customizable sun shader

  • C#
  • Blender
